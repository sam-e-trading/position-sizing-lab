# Contributing

Thanks for wanting to improve Position Sizing Game.

## Good contributions

- clearer educational explanations
- additional preset systems
- accessibility improvements
- mobile layout polish
- bug fixes in the simulator maths
- classroom/workshop features such as reset/share/export

## Keep it simple

This app is intentionally dependency-free. Please avoid adding a build system unless there is a strong reason.

## Financial advice boundary

Do not add trade recommendations, broker links, asset-specific calls, or claims that a preset is profitable in real markets. This is a risk education toy.

## Style

Clear, practical, slightly dry humour is welcome. Hype is not.
